# Notification Sounds

This folder stores frontend notification sounds used by `useNotificationSounds`.

Active files used by the app:
- `new-conversation.wav`
- `new-message.wav`
- `handoff.wav`

Optional placeholders (kept for compatibility with existing references):
- `new-conversation.mp3`
- `new-message.mp3`
- `handoff.mp3`

These files are intentionally frontend-only and are played with `HTMLAudioElement`.
